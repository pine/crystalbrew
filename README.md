crystalbrew
-----------

[Crystal](http://crystal-lang.org/) version manager.

## Install
One liner curl install.

```
$ curl -L j.mp/crystalbrew | perl - setup
```

Or, download and setup.

```
$ wget j.mp/crystalbrew
$ perl nodebrew setup
```
